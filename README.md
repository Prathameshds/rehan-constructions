# Rehan Constructions - Construction Management System

A comprehensive construction management system built for Rehan Constructions, located at 32 Shirala, Sangli, Maharashtra.

## Features

- **Deal & Project Management**: Track all construction deals, estimates, advances, and balances
- **Worker Management**: Manage worker payments, track paid and remaining amounts
- **Material Management**: Track material expenses, lease agreements, and payments
- **Financial Reports**: Generate detailed financial reports with profit/loss analysis
- **Advanced Filtering**: Filter data by week, month, year, or custom date ranges
- **Export Options**: Export reports to PDF and Excel formats

## Technology Stack

- HTML5, CSS3, JavaScript (ES6+)
- Firebase Realtime Database
- jsPDF for PDF generation
- SheetJS (XLSX) for Excel export
